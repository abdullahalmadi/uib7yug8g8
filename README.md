# مجلد المقالات 📝

مرحباً بك في مجلد المقالات! هنا يتم حفظ جميع مقالات المدونة.

## كيفية إضافة مقالة جديدة:

1. أنشئ ملف HTML جديد في هذا المجلد
2. اتبع الهيكل التالي في كل مقالة:

```html
<!DOCTYPE html>
<html lang="ar" dir="rtl">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="وصف المقالة الكامل">
    <meta name="keywords" content="كلمات مفتاحية، متعددة، هنا">
    <title>عنوان المقالة - مدونة عاشوايي</title>
    <link rel="stylesheet" href="../css/style.css">
</head>
<body>
    <!-- نفس شريط التنقل من الصفحة الرئيسية -->
    <nav class="navbar">
        <!-- محتوى التنقل -->
    </nav>
    
    <!-- محتوى المقالة -->
    <section class="posts">
        <div class="container">
            <h1>عنوان المقالة</h1>
            <!-- محتوى المقالة هنا -->
        </div>
    </section>
    
    <!-- التذييل -->
    <footer class="footer">
        <p>&copy; 2026 مدونة عاشوايي - جميع الحقوق محفوظة</p>
    </footer>
</body>
</html>
```

## نصائح مهمة:

✅ استخدم كلمات مفتاحية ذات صلة في كل مقالة
✅ تأكد من استخدام الاتجاه من اليمين إلى اليسار (RTL)
✅ استخدم أسماء واضحة للملفات
✅ أضف صور ووسائط لتحسين المقالات
✅ اكتب محتوى عالي الجودة وسهل الفهم
